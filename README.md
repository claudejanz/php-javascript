## Installation
1. Clone repository
2. Install docker
## Run applications
```bash
docker-compose up -d
```
## View result
- yii2 http://localhost:8080
- laravel http://localhost:8081
- symfony http://localhost:8082
- vuejs http://localhost:8083
- reactjs http://localhost:8084
- angularjs http://localhost:8085
